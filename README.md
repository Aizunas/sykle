# Sykle - E-Bike Rewards Platform

A cycling rewards app that converts bike rides into redeemable points at local caf√©s.

## Project Structure

- `/ios` - iOS app (Swift, SwiftUI, HealthKit)
- `/backend` - API server (Node.js, Express, SQLite)
- `/docs` - Documentation and design files

## Tech Stack

**iOS App:**
- Swift & SwiftUI
- HealthKit integration
- iOS 17+

**Backend:**
- Node.js & Express
- SQLite database
- RESTful API

## Getting Started

### Backend
```bash
cd backend
npm install
npm run dev
```

### iOS
1. Open `ios/Sykle.xcodeproj` in Xcode
2. Update IP address in `Config.swift`
3. Run on iPhone
